# authentication

1. token

   关于token,主要用了两个方法,一个作用在登录接口,账号密码验证成功以后会将用户的一部分信息用GetJWT()这个方法处理成jwt,然后将token返回给浏览器.二个是写了一个检验token的中间件,CheckJWT(),这个方法作用于需要验证token的接口,这个方法返回token的检验结果从而判断接下来的处理流程,检测主要检测两个部分,一个是token的合法性,一个是签发时间过没有.我预留了一个返回新的jwt的返回值(更新签发时间),所以这个方法既可以用来做中间件验证token的合法性,也可以单独作为一个更新jwt的接口,也可以与浏览器配合，作为中间件使用每次申请某个接口时对token进行更新,达到持续访问的效果.

   用了一个测试接口/test/token来测试返回的token,结果符合预期,token验证失败直接返回,token验证成功进入下一流程.
